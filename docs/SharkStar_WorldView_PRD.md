# 1. 世界观概述

## 1.1 核心叙事

在遥远的宇宙深处，有一颗名为「鲨之星」的星球。这颗星球承载着一位乒坛传奇的所有记忆、热爱与梦想——从石家庄到世界之巅的成长轨迹，都将在这里被永恒铭刻。

人类文明决定开发这颗星球，作为粉丝与偶像精神永恒连接的家园。

**为什么要来这个星球？**

因为地球上已经充满了争吵、暴力与不和谐。也没有一个地方能够保存所有粉丝与偶像专属的记忆，网络上都被AI生成的大量内容所充斥。而这颗星球，将成为热爱与真实记忆的永恒庇护所。

**第一批AI先遣队已经登陆。**

它们正在勘探地形、采集资源、建立基础设施。但AI机器人需要能源才能持续工作——这份能源，来自地球上每一位鲨鱼的热爱与参与。

## 1.2 世界观定位

| 维度       | 描述                                    |
| -------- | ------------------------------------- |
| **时间线**  | 近未来，地球已经被AI高等智慧所控制，一部分人类寻找保存自身文明的“基地” |
| **技术背景** | AI机器人先行开发，人类后续登陆                      |
| **核心冲突** | 多个粉丝星球竞争开发进度                          |
| **终极目标** | 建设人类与AI共存的理想乐园                        |
| **用户角色** | 星球开发的远程贡献者 → 未来的首批居民                  |

## 1.3 叙事价值

```markdown
用户心理转变：
"我在玩一个消除游戏" → "我在为我们的星球贡献能量"
"我在刷社交媒体" → "我在为AI机器人提供算力"
"我在攒积分" → "我在争取首批登陆资格"
```

**情感钩子**：
- **归属感**：这是「我们的」星球，不是「他们的」产品
- **使命感**：每一次游戏都有意义，都在推动星球发展
- **紧迫感**：其他星球也在竞争，我们不能落后
- **期待感**：未来能亲自登陆，决定星球的样貌

# 2. 星球开发叙事

## 2.1 开场动画脚本

**场景1：宇宙深空（3秒）**
- 画面：浩瀚星海，镜头缓缓推进
- 旁白：*“在距离地球4.2光年的宇宙深处…”*

**场景2：鲨之星出现（5秒）**
- 画面：一颗蓝色星球从黑暗中浮现，表面有微弱的光点
- 旁白：*“有一颗承载热爱与梦想的星球，正在苏醒”*

**场景3：AI机器人登陆（8秒）**
- 画面：飞船降落，舱门打开，Q版AI机器人（鲨鱼造型）列队走出
- 旁白：*“人类派出的第一批AI先遣队已经抵达”*
- 细节：机器人身上有「AMIO」标志，眼睛是蓝色光点

**场景4：机器人开始工作（6秒）**
- 画面：机器人开始建造、采集，但动作越来越慢
- 旁白：*“它们需要能量，才能继续建设这颗星球”*
- 视觉：机器人身上的能量条逐渐变红

**场景5：镜头拉回地球/手机（5秒）**
- 画面：从星球拉远，穿越星空，回到地球，最后落到用户手中的手机
- 旁白：*“而这份能量，来自每一位鲨鱼的热爱”*

**场景6：号召行动（3秒）**
- 画面：游戏界面闪现，能量飞向星空
- 旁白：*“每一次游戏，每一次分享，都在点亮鲨之星”*
- 文字浮现：**“首批登陆资格，等待你的争取”**

# 3. 双能源系统

## 3.1 能源类型定义

鲨之星的AI机器人需要两种能源才能运转：

| 能源类型 | 名称 | 来源 | 图标 | 叙事含义 |
|----------|------|------|------|----------|
| **行动能源** | ⚡ 动力核心 | 玩游戏获得 | 闪电图标 | AI的动力来源 |
| **智慧能源** | 💡 算力晶体 | 社交互动获得 | 灯泡图标 | AI的思考能力 |

## 3.2 行动能源（⚡ 动力核心）

**获取方式**：

| 行为 | 能源获取 | 说明 |
|------|----------|------|
| 通关普通关卡 | +100 ⚡ | 基础奖励 |
| 通关Hero模式 | +300 ⚡ | 高难度加成 |
| 宝箱等级加成 | ×1.0~2.0 | 钻石箱×2.0，金箱×1.5，银箱×1.2，铜箱×1.0 |
| 连续登录加成 | ×1.1~1.5 | 7天×1.1，14天×1.3，30天×1.5 |

**计算示例**：

```markdown
玩家A：通关Hero + 钻石箱 + 连续30天
= 300 × 2.0 × 1.5 = 900 ⚡

玩家B：通关普通 + 铜箱 + 首日
= 100 × 1.0 × 1.0 = 100 ⚡
```

## 3.3 智慧能源（💡 算力晶体）

**获取方式**：

| 行为 | 能源获取 | 说明 |
|------|----------|------|
| 分享到外部平台 | +50 💡 | 微信/微博/小红书等 |
| 邀请新用户注册 | +200 💡 | 被邀请者首次通关后到账 |

| 获得点赞 | +5 💡/赞 | 单帖上限100赞 |
| 获得评论 | +10 💡/条 | 单帖上限50条 |
| 参与话题讨论 | +20 💡 | 官方话题加成 |

**社交裂变机制**：

# 4. 星球开发进度机制

## 4.1 进度计算公式

```markdown
星球总进度 = f(全服⚡总量, 全服💡总量, 活跃用户数, 时间因子)

简化版：
进度增量/天 = (∑⚡ + ∑💡) × 活跃系数 × 时间衰减
```

**设计原则**：
- 早期进度快，给新用户成就感
- 中期稳定，需要社区共同努力
- 后期需要大事件推动（如赛事联动）


**里程碑事件**：

# 6. 首批登陆者权益

## 6.1 登陆资格计算

**综合贡献值**：

```markdown
贡献值 = ⚡动力核心 × 0.6 + 💡算力晶体 × 0.4 + 早期加成
```

**早期加成**：

| 注册时间 | 加成系数 |
|----------|----------|
| 内测期 | ×1.5 |
| 公测首月 | ×1.3 |
| 公测3月内 | ×1.2 |
| 公测6月内 | ×1.1 |
| 之后 | ×1.0 |

## 6.2 登陆批次与权益

| 批次 | 资格条件 | 人数 | 专属权益 |
|------|----------|------|----------|
| **首批先驱者** | 贡献值 Top 1% | ~150人 | 永久星球风格决定权 + 专属地标冠名 + 限定手办 |
| **早期开拓者** | 贡献值 Top 5% | ~750人 | 参与风格投票 + 专属建筑 + 限定周边 |
| **建设者** | 贡献值 Top 20% | ~3000人 | 优先登陆 + 专属称号 |
| **普通居民** | 完成基础任务 | 不限 | 正常登陆 |

## 6.3 首批登陆者的风格决定权

**机制说明**：

当星球进度达到100%，3D虚拟世界开放前，首批登陆者将参与决定：

| 决定项 | 选项示例 | 影响范围 |
|--------|----------|----------|
| **星球整体风格** | 科幻风 / 童话风 / 复古风 | 所有建筑视觉 |
| **核心建筑设计** | 投票选择3D模型方案 | 主要地标外观 |
| **背景音乐风格** | 电子 / 轻音乐 / 史诗 | 全局BGM |
| **NPC AI性格** | 活泼 / 稳重 / 幽默 | AI机器人对话风格 |

**投票权重**：

```markdown
首批先驱者：5票
早期开拓者：3票
建设者：1票
```

## 6.4 专属地标冠名

**Top 100贡献者**可获得地标冠名权：

## 10.2 实时同步需求

| 数据类型  | 更新频率 | 同步方式         |
| ----- | ---- | ------------ |
| 星球总进度 | 每5分钟 | 轮询/WebSocket |

| 排行榜   | 每小时  | 定时刷新         |
| 里程碑事件 | 触发时  | 推送通知         |

## 10.3 3D虚拟世界预留

**技术选型考量**：

| 方案 | 优势 | 劣势 | 推荐场景 |
|------|------|------|----------|
| Unity WebGL | 3D能力强 | 包体大，加载慢 | 完整虚拟世界 |
| Three.js | 轻量，Web原生 | 复杂场景性能差 | 简单3D预览 |
| Cocos Creator | 跨平台好 | 3D能力弱于Unity | 小程序兼容 |

**MVP阶段**：
- 使用2D动画模拟星球视觉变化
- 预留3D数据接口
- 登陆后再接入完整3D世界
