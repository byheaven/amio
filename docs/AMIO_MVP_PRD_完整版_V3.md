# 第一部分：通用游戏架构

> 本部分定义所有偶像星球共用的系统框架，是AMIO平台的核心骨架。
> 接入新偶像时，只需配置「第二部分」的个性化内容。

---

## 1. 产品概述

### 1.1 一句话定义

**每天一关偶像主题消除游戏，通关越快宝箱越好，挑战Hero模式赢实体周边。**

### 1.2 目标用户

- 核心用户：偶像粉丝群体（18-35岁为主）
- 特征：愿意为偶像付出时间和金钱，活跃于粉丝社群
- 需求：与偶像产生连接感，获得专属感和归属感

### 1.3 核心价值主张

| 价值点 | 说明 |
|--------|------|
| **专属感** | AI定制的专属游戏体验 |
| **连接感** | 通过剧情了解偶像成长故事 |
| **获得感** | 真实的实体周边奖励 |
| **成就感** | Hero模式的挑战与荣耀 |

### 1.4 商业模式

| 阶段 | 变现方式 |
|------|----------|
| MVP | 激励广告（道具获取） |
| V2 | 周边销售、抽奖付费 |
| V3 | 会员订阅、联名合作 |

## 3. 3-Tiles 玩法系统

### 3.1 玩法概述

**类型**：堆叠消除 / Tile-Matching / 3-Tiles

**核心规则**：
- 点击选中图块 → 移入底部槽位
- 槽位凑齐3个相同图块 → 自动消除
- 消除所有图块 → 通关
- 槽位满且无法消除 → 失败

### 3.2 游戏界面布局

```
┌─────────────────────────────────────────┐
│  [返回]    今日关卡 Day.47    [设置]    │
├─────────────────────────────────────────┤
│                                         │
│         ┌─────────────────┐             │
│         │                 │             │
│         │   [图块区域]     │             │
│         │                 │             │
│         │  层叠的主题图块  │             │
│         │  只能点最上层    │             │
│         │                 │             │
│         └─────────────────┘             │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│  槽位：[1][2][3][4][5][6][7]            │
│                                         │
│  凑齐3个相同 → 自动消除                  │
│  槽位满了 → 游戏失败                     │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│  道具：[撤回 1/1] [移出 0/1] [洗牌 0/1] │
│                                         │
└─────────────────────────────────────────┘
```

### 3.3 图块设计规范

| 属性 | 规范 |
|------|------|
| 尺寸 | 60×60 像素（@2x） |
| 形状 | 圆角正方形，圆角半径8px |
| 层叠 | 上层图块遮挡下层，下层显示阴影 |
| 可点击 | 最上层图块高亮，下层图块灰暗 |
| 动画 | 点击时缩放0.9→1.0，移入槽位有轨迹动画 |

### 3.4 槽位机制

| 项目 | 设定 |
|------|------|
| 槽位数量 | 7格（固定） |
| 放入规则 | 新图块放入最左侧空位 |
| 消除判定 | 任意3个相同图块相邻即消除 |
| 消除动画 | 3个图块合并→放大→消失，0.3秒 |
| 失败判定 | 第7格被占用且无法消除 |

### 3.5 道具系统

**道具列表**：

| 道具 | 图标 | 效果 | 每局次数 | 获取方式 |
|------|------|------|----------|----------|
| 撤回 | ↩️ | 将最近放入的1个图块退回原位 | 1次 | 免费 |
| 移出 | 📤 | 将槽位中3个图块移出暂存区 | 1次 | 看广告/50金币 |
| 洗牌 | 🔀 | 随机打乱所有未选中图块位置 | 1次 | 看广告/50金币 |

**道具使用规则**：

| 规则 | 说明 |
|------|------|
| 使用时机 | 游戏进行中任意时刻 |
| 次数限制 | 每种道具每局限用1次 |
| 暂存区 | 移出的图块显示在侧边，需手动放回 |
| 统计影响 | 使用道具会影响宝箱等级评定 |

**道具交互流程**：

```
点击道具按钮
    │
    ├─→ 有免费次数？──→ 直接使用 ──→ 扣减次数
    │
    └─→ 无免费次数？──→ 弹出选择
                           │
                    ┌──────┴──────┐
                    ↓             ↓
                看广告(30秒)   花费50金币
                    │             │
                    └──────┬──────┘
                           ↓
                       使用道具
```

## 4. 每日关卡系统

### 4.1 关卡刷新机制

| 项目 | 设定 |
|------|------|
| 刷新时间 | 每日 00:00（服务器时间） |
| 关卡数量 | 每天1个全新关卡 |
| 挑战次数 | 无限（直到通关） |
| 关卡保存 | 不保存中途进度，每次从头开始 |

### 4.2 关卡难度设计

**难度目标**：60%用户可在5次内通关

| 参数 | 普通模式 | 说明 |
|------|----------|------|
| 图块种类 | 8-10种 | 从偶像主题元素库中选取 |
| 图块总数 | 72-90个 | 必须是3的倍数 |
| 堆叠层数 | 3-4层 | 控制策略深度 |
| 布局类型 | 随机生成+人工微调 | 确保有解 |

**关卡生成规则**：

1. 图块总数必须是3的倍数（确保理论可解）
2. 每种图块数量必须是3的倍数
3. 顶层图块至少有3种不同类型（避免开局卡死）
4. 服务端预生成并验证可解性

### 4.3 关卡与剧情关联

| 关卡日期 | 剧情关联 |
|----------|----------|
| Day 1-7 | 第一章剧情 |
| Day 8-14 | 第二章剧情 |
| Day 15-21 | 第三章剧情 |
| ... | ... |

每完成一定天数的关卡，解锁对应章节的剧情动画。

## 5. 宝箱系统

### 5.1 宝箱等级

| 等级 | 图标 | 名称 | 视觉风格 |
|------|------|------|----------|
| Lv.4 | 💎 | 钻石宝箱 | 钻石材质，闪耀光芒 |
| Lv.3 | 🥇 | 黄金宝箱 | 金色材质，华丽装饰 |
| Lv.2 | 🥈 | 白银宝箱 | 银色材质，简洁优雅 |
| Lv.1 | 🥉 | 青铜宝箱 | 青铜材质，朴素外观 |

### 5.2 宝箱等级评定

**评定公式**：基于挑战次数 + 道具使用数

| 等级 | 挑战次数 | 道具使用 | 综合条件 |
|------|----------|----------|----------|
| 💎 钻石 | 1次 | 0个 | 首次通关且不用道具 |
| 🥇 黄金 | 1-2次 | ≤1个 | 快速通关少用道具 |
| 🥈 白银 | 3-5次 | ≤2个 | 正常通关 |
| 🥉 青铜 | 6次+ | 任意 | 艰难通关 |

**评定优先级**：挑战次数 > 道具使用

**详细评定表**：

| 挑战次数 | 0道具 | 1道具 | 2道具 | 3道具 |
|----------|-------|-------|-------|-------|
| 1次 | 💎 | 🥇 | 🥈 | 🥈 |
| 2次 | 🥇 | 🥇 | 🥈 | 🥈 |
| 3次 | 🥈 | 🥈 | 🥈 | 🥉 |
| 4-5次 | 🥈 | 🥈 | 🥉 | 🥉 |
| 6次+ | 🥉 | 🥉 | 🥉 | 🥉 |

### 5.3 宝箱解锁机制

**核心规则**：今天玩，明天开

| 规则 | 说明 |
|------|------|
| 锁定时间 | 获得后24小时解锁 |
| 领取窗口 | 解锁后24小时内可领取 |
| 过期处理 | 超时未领取则宝箱消失 |
| 最大持有 | 同时最多1个待领取宝箱 |

**时间线示例**：

```
Day 1 20:00 - 通关获得宝箱（锁定中）
Day 2 20:00 - 宝箱解锁，可领取
Day 3 20:00 - 若未领取，宝箱过期消失
```

## 6. Hero模式

### 6.1 概述

Hero模式是通关普通关卡后的**可选**高难度挑战，提供风险与回报的博弈。

| 项目 | 说明 |
|------|------|
| 解锁条件 | 当日普通关卡通关 |
| 挑战次数 | 仅1次机会 |
| 失败惩罚 | 无（保持原宝箱等级） |
| 通关奖励 | 宝箱升2级 + 实体概率×3 |

### 6.2 难度对比

| 参数 | 普通模式 | Hero模式 | 提升幅度 |
|------|----------|----------|----------|
| 图块种类 | 8-10种 | 12-15种 | +50% |
| 图块总数 | 72-90个 | 108-135个 | +50% |
| 堆叠层数 | 3-4层 | 5-6层 | +50% |
| 预计通关率 | 60% | 20% | -67% |

### 6.3 Hero奖励机制

**宝箱升级**：

| 原宝箱等级 | Hero通关后 | 升级幅度 |
|------------|-----------|----------|
| 🥉 青铜 | 🥈 白银 | +2级（但不超过白银） |
| 🥈 白银 | 🥇 黄金 | +2级（但不超过黄金） |
| 🥇 黄金 | 💎 钻石 | +1级 |
| 💎 钻石 | 💎 钻石+ | 保持+额外奖励 |

**实体周边概率提升**：

| 宝箱等级 | 普通实体概率 | Hero通关后 |
|----------|-------------|------------|
| 💎 钻石+ | 10% | 30%（×3） |
| 💎 钻石 | 10% | 30%（×3） |
| 🥇 黄金 | 3% | 9%（×3） |
| 🥈 白银 | 0% | 3%（新增） |
