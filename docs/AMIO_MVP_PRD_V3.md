# AMIO MVP PRD

## AI为爱定制 · 玩游戏，赢周边

---

## 一句话说清楚

**每天一关偶像主题消除游戏，通关越快宝箱越好，挑战Hero模式赢实体周边。**

---

## 核心循环

```markdown
每日关卡 → 通关获得宝箱 → [可选] Hero模式 → 次日开箱 → 拿奖励
    ↑                         ↓
    └──── 想要更好的宝箱/周边 ←┘
```

---

## 玩法：3-Tiles 堆叠消除

### 基础规则

```markdown
┌─────────────────────────────┐
│                             │
│    [层叠的主题图块]          │
│     只能点击最上层           │
│                             │
├─────────────────────────────┤
│  槽位：[ ][ ][ ][ ][ ][ ][ ] │
│        （最多7格）           │
│  凑齐3个相同 → 自动消除      │
│  槽位满了 → 失败             │
└─────────────────────────────┘
```

### 道具（每局各1次）

| 道具 | 效果 | 获取 |
|------|------|------|
| 撤回 | 退回最近1个图块 | 免费1次 |
| 移出 | 移走槽位中3个图块暂存 | 看广告/金币 |
| 洗牌 | 打乱未选图块位置 | 看广告/金币 |

## 每日关卡设计

### 关卡难度

| 对比 | 羊了个羊 | 鲨之星 |
|------|----------|--------|
| 难度 | 极难（<0.1%通关） | 适中（目标60%通关） |
| 体验 | 靠运气 | 靠策略+少量运气 |
| 目的 | 逼看广告 | 鼓励每日回访 |

### 每日更新

- 每天0点刷新1个新关卡
- 关卡主题与偶像剧情关联
- 可无限次挑战，直到通关

## 宝箱系统（核心）

### 宝箱等级计算

**公式：挑战次数越少 + 道具用得越少 = 等级越高**

| 等级 | 条件 | 宝箱外观 |
|------|------|----------|
| 💎 钻石 | 1次通关 + 0道具 | 闪耀钻石箱 |
| 🥇 黄金 | 1-2次通关 + ≤1道具 | 金色宝箱 |
| 🥈 白银 | 3-5次通关 + ≤2道具 | 银色宝箱 |
| 🥉 青铜 | 6次以上或3道具全用 | 青铜宝箱 |

### Hero模式（通关后解锁）

**通关普通关卡后，可选择挑战Hero模式：**

| 项目 | 普通模式 | Hero模式 |
|------|----------|----------|
| 难度 | 适中 | 困难 |
| 图块层数 | 3-4层 | 5-6层 |
| 图块种类 | 8-10种 | 12-15种 |
| 机会 | 无限 | 仅1次 |

**Hero模式奖励：**

| 结果 | 效果 |
|------|------|
| 通关 | 宝箱升2级 + 实体周边概率×3 |
| 失败 | 无惩罚，保持原宝箱等级 |

### 宝箱规则

| 规则 | 说明 |
|------|------|
| 今天玩，明天开 | 宝箱需24小时解锁 |
| 不来领，就作废 | 超24小时未领取过期 |
| 连续14天必得周边 | 贴纸/明信片包邮到家 |

## 奖励内容

### 宝箱奖励池

| 等级 | 奖励内容 |
|------|----------|
| 💎 钻石 | 金币×500 + 道具×3 + 抽奖券×2 + 10%实体周边 |
| 🥇 黄金 | 金币×300 + 道具×2 + 抽奖券×1 + 3%实体周边 |
| 🥈 白银 | 金币×150 + 道具×1 + 5%抽奖券 |
| 🥉 青铜 | 金币×50 + 道具碎片 |

### Hero模式额外奖励

| 原等级 | Hero通关后 | 实体概率 |
|--------|-----------|----------|
| 青铜→白银 | +2级 | ×3 |
| 白银→黄金 | +2级 | ×3 |
| 黄金→钻石 | +1级（封顶） | ×3 |
| 钻石→钻石+ | 保持+额外奖励 | ×5 |

### 连续开箱保底

| 连续天数 | 保底奖励 |
|----------|----------|
| 7天 | 抽奖券×3 |
| 14天 | 实体贴纸包（包邮） |
| 30天 | 实体手链/挂件（包邮） |

## AI定制（差异化）

用户可用AI改变：
- 图块外观风格（可爱/酷炫/复古）
- 整体配色主题
- 背景音乐风格

**每个人玩的都是自己创作的专属游戏。**

## 剧情推进

- 每日关卡对应偶像成长故事的一个片段
- 通关后解锁15-30秒剧情动画
- 累计通关推进整体剧情线
