# Tab Bar Icons Required

Taro tabBar requires 81x81px PNG images with transparency.

Required icons:
- `starlight.png` / `starlight-active.png` - Star icon for 星光 tab
- `starocean.png` / `starocean-active.png` - Wave/ocean icon for 星海 tab
- `startrail.png` / `startrail-active.png` - Orbit/path icon for 星轨 tab

The app will work without these icons (showing text labels only).

To add icons:
1. Create 81x81px PNG files
2. Place them in this directory
3. Update app.config.ts to reference them
